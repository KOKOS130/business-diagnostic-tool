# 事業推進力診断ツール

経営者向けの事業推進力を診断するWebアプリケーションです。

## 機能

- 6つの軸（36問）で事業推進力を総合診断
- リアルタイムでレーダーチャート表示
- A/B/C/Dランク判定
- 具体的な改善アクションの提案

## 診断軸

1. 経営ビジョンの明確さ (6問)
2. 事業計画の実行管理 (7問)
3. 組織体制の強さ (6問)
4. 経営者の時間の使い方 (6問)
5. 数値管理の仕組み (6問)
6. 収益性の健全度 (6問)

## 使い方

### ローカルで実行

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

### Webで公開

Streamlit Cloudで公開可能です。
